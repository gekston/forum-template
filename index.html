Free Material Design Template

By Wiebe Dijkstra

Hello,

I developed a Storyline template based on Material Design by Google.

Features

  • Floating menu (like in an app)
  • Progress bar
  • 3 Question types
  • UI elements (buttons, radio button, check box)
  • Custom Feedback Master

Enjoy!
